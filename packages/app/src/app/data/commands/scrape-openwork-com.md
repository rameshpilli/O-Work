name: scrape-openwork-com
description: Verify browser prerequisites and scrape data from openwork.com
---

Open your browser on `https://openwork.com` and scrape the relevant page data.

Requirements:
1. Before any browser automation, verify that `npx` is installed on this machine.
2. If `npx` is missing, install the required Node/npm tooling first and confirm the command works.
3. Prefer the available browser automation path; if browser tooling needs setup, do that work instead of stopping early.
4. Open `openwork.com`, scrape the page data, and return the extracted results in a clean list.
5. Finish only after the browser flow has actually run and you have the scraped output.
